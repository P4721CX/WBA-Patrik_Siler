WORK

These are some of my projects that I've been working on.

When I was little, I've been editing some funny videos with my friends. I've done them in programmes like Camtasia and Filmora. Nowadays, I don't do these any more, but after this period of editing videos I started drawing in programmes such as Medibang Paint Pro and Krita.

Now I'm more into pixel art. In my opinion, pixel art style is better because it's easier to make and also 'cause I can use them in my games that I make, which is my newest passion.

I haven't done any full game yet, however I've got a few in progress. The only game I made in Unity was my Meteor Strike, which is a total copy of Sebastian Lague's game called Falling Blocks.

Right now, I want to get better at programming, so I can make some kind of popular game, I want to learn with Arduino and finally get better at coding websites.
